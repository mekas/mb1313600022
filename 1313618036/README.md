## Left-Store
Left-Store is an application where users can find information of local shoes in Indonesia

## Target Segment
- Users who like local shoes
- Left-Store Visitor

## Advanced Topic
- Rest API
- Web View

## Feature
- List of local shoes
- Detail of Local Shoes
- News of Local Shoes
- Add to Favourite
